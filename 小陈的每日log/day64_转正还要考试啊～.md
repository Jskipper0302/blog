### 学习
这次 ts axios 重构的学习着实难了许多，平常使用 axios 的时候，可以

```javascript
axios({
	method: 'post',
  url: 'xxxxxxx',
  params: {}
})
```

也可以

```javascript
axios.post(url, {
  params: {}
})
```

这里就要求 axios 自己本身是一个方法，但他也有属性方法。
在 ts 中可用混合类型、混合接口、交叉类型实现，这个地方得好好研究透了才行。
